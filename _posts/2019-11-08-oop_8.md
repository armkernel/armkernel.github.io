### Intro
- State pattern에 대해 알아보자.

#### 메이자스토리를 생각해보자
- 메이플스토리 사진

- 위 사진을 보면 하나의 케릭터가 많은 일을 한다는 것을 알 수 있다. (e.g. 공격, 달리기, 아이템 줍줍 등)
  - 그렇다면, 이런 케릭터는 어떻게 객체로 표현할 것인가?

#### 단순한 케릭터 객체

```c
class Character
{
public:
  void run() { cout << "run" << endl; }
  void attack() { cout << "attack" << endl; }
};

int main()
{
  Character* c;
  c->run();
  c->attack();
}
```
